# ReadMe #

A small Sudoku solver. Will solve any possible sudoku problem, even an empty grid.

Includes a basic Swing GUI.


# Download #

[Runnable .jar file](https://bitbucket.org/twhitty/sudoku/downloads/SudokuSolver_ver_2_1.jar)
